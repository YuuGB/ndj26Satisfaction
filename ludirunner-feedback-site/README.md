# Ludirunner Feedback Site

This project is designed to collect and analyze public satisfaction during the Ludirunner game night event. The website allows users to provide feedback on their experience and displays aggregated results for analysis.

## Project Structure

```
ludirunner-feedback-site
├── public
│   └── index.html        # Main HTML structure for collecting user feedback
├── src
│   ├── app.js           # Main JavaScript file for handling feedback logic
│   ├── styles.css       # CSS styles for the website
│   └── data
│       └── survey.js    # Functions for managing survey data
├── package.json          # npm configuration file
└── README.md             # Documentation for the project
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ludirunner-feedback-site
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the site.

## Functionality

- Users can submit feedback regarding their experience during the Ludirunner game night.
- The site collects responses and provides visualizations of the aggregated results.
- Admins can analyze the feedback to improve future events.
